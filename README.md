# Book Store GitHub Pages

เว็บไซต์ Static HTML/CSS/JavaScript พร้อมอัปโหลดขึ้น GitHub Pages

## วิธีใช้งาน
1. สร้าง Repository ใหม่ใน GitHub
2. อัปโหลด `index.html`, `style.css`, `script.js` และโฟลเดอร์ `assets`
3. ไปที่ Settings > Pages
4. เลือก Deploy from a branch
5. เลือก branch `main` และโฟลเดอร์ `/root`
6. กด Save แล้วรอ GitHub สร้างลิงก์เว็บไซต์
